Course Access Codes

To use a Course Access Code, you must have a cultivatelearning.ca account, and be signed in.
Register or sign in.

If you received a code-embeded link via email, follow it for code-less entry.
To enter a code manually, sign in and:

  1. Click the Unlock a course using an access code button located on the My Courses page.
  2. Copy and paste the Course Acccess Code you received and click Enter.

Purchased and unlocked courses

All your courses are available from the the top-right corner's User Menu dropdown under My Courses.
Sign in to access it.

Inquiries about a specific webcourse

Please contact the corresponding publisher.
Details are available in the Publisher tab inside each course, and on course specific pages, accessible through following the course title link.
